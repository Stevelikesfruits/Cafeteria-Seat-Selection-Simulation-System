## C4 Level 3：组件交互时序 (关键 Tick 流程)

```mermaid
sequenceDiagram
    participant Timer as QTimer
    participant MW as MainWindow
    participant SE as SimulationEngine
    participant SG as StudentGenerator
    participant SA as SeatAllocator
    participant SC as SatisfactionCalculator
    participant R as Restaurant
    participant RV as RestaurantView

    Note over Timer,RV: 每 500ms 触发一次 (仿真 1 分钟)
    
    Timer->>MW: timeout 信号
    MW->>SE: step()
    
    activate SE
    Note over SE: current_time += 1
    
    rect rgb(255,230,230)
        Note over SE,R: 阶段1: 处理离店
        SE->>SE: _process_departures()
        SE->>R: 清空到期学生的 seat(i)=0
    end
    
    rect rgb(230,255,230)
        Note over SE,SC: 阶段2: 生成新学生
        SE->>SG: generate_batch(current_time, pref_ratios)
        SG-->>SE: List(Student)
    end
    
    rect rgb(230,230,255)
        Note over SE,R: 阶段3: 分配座位
        loop 每个新生
            SE->>SA: allocate(student)
            SA->>R: 遍历所有 Table, 查找空位
            alt 找到完美匹配
                SA->>R: seats(idx) = student.id
                SA-->>SE: True (完美)
            else 降级到任意空位
                SA->>R: seats(idx) = student.id
                SA-->>SE: True (降级)
            else 无空位
                SA-->>SE: False
            end
            SE->>SC: calculate(student, is_perfect_match)
            SC-->>SE: 写入 satisfaction_score
        end
    end
    
    deactivate SE
    
    MW->>RV: update_view(restaurant)
    RV->>R: 读取所有 tables(i).seats()
    RV->>RV: 更新椅子颜色 (空=蓝 / 有人=金)
```

## 容器间接口契约

| 调用方 | 被调用方 | 接口 | 数据方向 |
|--------|----------|------|----------|
| MainWindow | SimulationEngine | `engine.step()` | 无参数 |
| MainWindow | SimulationEngine | `engine.reset(n2, n4)` | UI --> Core (int*2) |
| MainWindow | SimulationEngine | `engine.get_statistics() --> dict` | Core --> UI |
| ControlPanel | SimulationEngine | `preferences_changed.emit(dict)` | UI --> Core (Qt Signal) |
| SimulationEngine | StudentGenerator | `generate_batch(time, ratios) --> List(Student)` | Core 内部 |
| SimulationEngine | SeatAllocator | `allocate(student) --> bool` | Core 内部 |
| SimulationEngine | SatisfactionCalculator | `calculate_and_assign(student, bool)` | Core 内部 (静态) |
| MainWindow | RestaurantView | `update_view(restaurant)` | UI 内部 |
| RestaurantView | Restaurant | 读取 `tables(id).seats()` | UI --> Models (只读) |
```
